# E-Commerce Customer Behavior Analysis

## Project Overview
This project analyzes customer purchasing behavior using Python and Jupyter Notebook. The goal is to understand customer demographics, spending patterns, membership types, and customer satisfaction.

## Dataset
The dataset contains 350 customer records with the following features:
- Customer ID
- Gender
- Age
- City
- Membership Type
- Total Spend
- Items Purchased
- Average Rating
- Discount Applied
- Days Since Last Purchase
- Satisfaction Level

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning
- Checked missing values
- Filled missing Satisfaction Level values using the mode
- Verified dataset consistency

## Exploratory Data Analysis
Created visualizations for:
- Gender Distribution
- Age Distribution
- Membership Type Distribution
- Customers by City
- Average Spending by Membership Type
- Correlation Heatmap
- Average Rating vs Total Spend

## Business Insights
- Gold members spend the most.
- Higher customer ratings are associated with higher spending.
- Customer spending varies across different cities.
- Missing values were successfully handled.

## Conclusion
This project demonstrates how data analysis can help businesses understand customer behavior and make better business decisions.
